# ABC-Анализ списаний товаров

Веб-приложение для проведения ABC-анализа данных о списаниях товаров из Excel-файлов.

## Основные возможности
- **Автоматизация:** Обработка файлов .xlsx и .xls.
- **Интерактивность:** Фильтрация по складам, категориям и поиск по названию.
- **Визуализация:** Построение Парето-диаграммы (принцип 80/20).
- **Экспорт:** Возможность выгрузки результатов анализа обратно в Excel.

## Как пользоваться
1. Откройте приложение.
2. Загрузите файл выгрузки из вашей учетной системы (МойСклад, 1С и т.д.).
    - Программа ожидает: Товар в столбце **C**, Склад в **D**, Кол-во в **F** (списания должны быть < 0).
3. Изучите вкладки "Общий", "По складам" и диаграмму Парето.
4. При необходимости отфильтруйте данные и нажмите "Экспорт Excel".

## Технологии
- HTML5, CSS3, JavaScript (Vanilla)
- [SheetJS (xlsx)](https://sheetjs.com/) — для парсинга Excel.
- [Chart.js](https://www.chartjs.org/) — для построения графиков.
